== Description ==

Generate wordpress pages, posts, categories, comments for development enviroment in few clicks.

You can generate entire site content in one click, or configure each entity and generate it separately.

You will be able to generate: categories, posts, pages.

[Video on YouTube showing how it works](http://www.youtube.com/watch?v=6pMOfc28Pc0&feature=player_embedded)

Featured images generator added in 1.3, but be patient if you generate many posts it takes some time to generate featured images, wordpress also creates thumbnails foreach image. To exclude timeout error you may change execution limit for php scripts in your php.ini.

Post custom meta fields generator added in 1.4, now you can generate anything you want.

== Installation ==

1. Unzip and upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
