== Description ==
<p>Allows viewing of next/prev post with keyboad navigation on single post pages. Also preloads and prerenders pre/next post page for chrome and firefox</p>

= Features =
* View next post/posts with "Pagedown" or "Right", previous post/posts with "Pageup" or "Left" on single post pages (of same category)
* 


== Installation ==
1. Download the Keyboard Nav Plus Preload zip file.
2. Extract the files to your WordPress plugins directory.
3. Activate the plugin via the WordPress Plugins tab.
